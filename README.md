# Tarockregeln auf Basis von Efelo und der FSTM

Eine HTML-basierte Regelreferenz für **Königsrufen** (Österreichisches Tarock), entstanden auf Grundlage der offiziellen kanonisierten Regeln der [FSTM](https://fstm.at/) (Fachschaft Technische Mathematik an der Technischen Universität Wien).

Danke an Efelo fürs Aufschreiben der ursprünglichen Regeln.

## Inhalt

| Datei | Beschreibung |
|---|---|
| `index.html` | Einstiegsseite (leitet weiter auf `Tarock.html`) |
| `Tarock.html` | Hauptseite: Grundregeln, Glossar und alle Spielvarianten |
| `faq.html` | Häufig gestellte Fragen |
| `style.css` | Zentrales Stylesheet für alle Seiten |
| `karten/Bild*.html` | Popup-Fenster mit Kartenbildern (Gstieß, Mond, Wildsau, Kakadu, Uhu, Pagat) |
| `Bsp*.html` | Beispielrechnungen (Abrechnung, Lizitation, Spielverlauf) |
| `bilder/` | Kartenbilder als JPG-Dateien |

## Lokal öffnen

Die Seite funktioniert ohne Server — einfach `index.html` im Browser öffnen.

## Lizenz

Die Inhalte basieren auf den Regeln der FSTM und werden ohne Gewähr zur Verfügung gestellt.
